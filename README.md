# Réussir son Entretien d'Embauche — Responsive

Version mobile-first avec 20 slides.

## Lancer
Ouvrez `index.html` dans un navigateur.

## Fonctionnalités
- Layout responsive (mobile/tablette/desktop)
- Navigation tactile (swipe), clavier, boutons
- Bouton **Plein écran** (⛶)
